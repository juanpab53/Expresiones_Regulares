
Desarrollar y aplicar expresiones regulares en Java para la validación de diversos formatos de datos comunes.

Se evaluará la correcta formulación de cada expresión regular, la comprensión de su funcionamiento, la adecuada implementación en Java.

Las expresiones regulares que contiene el programa funcionan para la validación de las siguientes entradas:

    1.      Placas de carro en Colombia.
    2.      Códigos postales para Medellín.
    3.      Números de teléfonos fijos en Medellín.
    4.      Números de teléfonos celulares en Colombia.
    5.      Direcciones de correo electrónico (formato general).
    6.      Fecha en formato dd/mm/aaaa (ej. 01/01/2023).
    7.      Hora en formato de 24 horas hh:mm:ss (ej. 14:30:00).
    8.      Hora en formato de 12 horas hh:mm:ss (ej. 02:30:00 AM/PM).
    9.      Declaración de variables en Java (ej. int contador = 0;).
    10.     URL (formato general).
    11.     Código ISBN (especificar si es ISBN-10 o ISBN-13).
    12.     Código de barras en Colombia.
    13.     Nombres y apellidos con tildes y 'ñ' (considerar mayúsculas y minúsculas).
    14.     Contraseña segura (definir criterios de seguridad).
    15.     Estructura de un ciclo for (ej. For(k=1; k<N;k++)).
    16.     Código para representar un número double (ej. 3.14159).
    17.     Leyes, decretos y artículos en Colombia.
    18.     Referencias de libros impresos en normas IEEE.
    19.     Referencias de libros impresos en normas APA.
    20.     Precios de productos.
    21.     Diptongos e hiatos.
